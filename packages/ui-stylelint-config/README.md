## ui-stylelint-config

A shared stylelint config made by Racoon UI

### Installation

```sh
yarn add @racoon-ui/ui-stylelint-config
```

### Usage

In your stylelint config file:

```js
/* stylelint.config.js */
module.exports = require('@racoon-ui/ui-stylelint-config')
```
